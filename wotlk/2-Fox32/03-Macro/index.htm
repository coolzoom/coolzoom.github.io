宏命令指南(Macro)¶

WLK单机版全角色通用的常用宏命令¶

新建角色测试通用¶

  1. 新人 - 在Fox3.2端中, 学习60%陆地坐骑, 100%陆地坐骑, 150%飞行坐骑都不要钱。而280%坐骑需要3000金, 诺森德飞行需要300金币

    /target player
.modify money 50000000 // 增加5000金币
.add 23162 4  // 增加4个36格包包
/targetlasttarget

  2. 飞起 - 所有地区通用坐骑: 安其拉飞行坦克

    /target player
.aura 31700 乘坐其拉飞行坦克
/targetlasttarget

  3. 飞落 - 离开坐骑, 给予4倍移动速度

    /target player
/dismount 下坐骑
.modify speed all 1
.unaura 39870 暴速
.aura 39870 暴速
/targetlasttarget

  4. X01 - 移动速度x1, 恢复正常速度

    /target player
.modify speed all 1
/targetlasttarget

  5. X03 - 移动速度x3

    /target player
.modify speed all 3
/targetlasttarget

  6. X16 - 移动速度x16

    /target player
.modify speed all 16
/targetlasttarget

  7. X32 - 移动速度x32

    /target player
.modify speed all 32
/targetlasttarget

  8. 加速 - 移动速度x2, 被打晕后不会出错

    /target player
.unaura 62375

常用特殊宏命令¶

  1. 刷怪 - 附近所有怪物刷新

    /cleartarget
.respawn

  2. On - 伤害免疫模式(但会吸引仇恨)

    /target player
.learn 39804 伤害免疫:魔法
.learn 34311 伤害免疫:物理
/targetlasttarget

  3. Off - 关闭伤害免疫模式

    /target player
.unlearn 39804 伤害免疫:魔法
.unlearn 34311 伤害免疫:物理
/targetlasttarget

  4. 假死 - 假死并脱离战斗状态

    /target player
.aura 5384 假死
/targetlasttarget

  5. 冷却 - 所有技能CD冷却

    /target player
.cooldown
/targetlasttarget

  6. 清除 - 清除身上所有debuff并且无敌, 再按一下解除无敌

    /target player
.aura 45438
.unaura 45438
/targetlasttarget

  7. 隐身 - 瞬间隐身并脱离战斗

    /target player
.npcbot command stay
.aura 32612 隐身

  8. 复活 - 复活当前玩家

    /g .revive

  9. 重置 - 所有副本重置

    .instance unbind all

  10. DB - 打开数据库查询

    /itemdb

其他GM宏命令¶

  • 添加NPC: 在玩家所在位置添加NPC, 通常是在地图上增加怪物群。

    /target player
.npc add 22313 地心震动者

  • 设置NPC移动方式: 将选定NPC的移动方式修改为随机走动。并设置刷新时间为1分钟

    /target 地心震动者
.npc set spawndist 20
.npc set movetype random
.npc set spawntime 60
