Addons¶

要玩好魔兽世界, 怎么能没有插件? 以下列出了个人经过大量比较和试用之后, 个人最喜欢的实用插件列表和简要说明。

  • AckisRecipeList - 商业技能配方资料库
  • Altoholic - 全角色物品信息统计
  • AtlasLoot - 副本装备掉落查询
  • AtlasQuest - 副本相关任务提示
  • Bagnon - 背包整合
  • BigWigs - TBC副本首领模块
  • BugSack - 错误信息隐藏
  • Combuctor - 背包整合插件
  • DBM - 副本首领模块
  • Domino - 多功能综合动作条
  • Engravings - 装备出处插件
  • EveryQuest - 未做完的任务查询
  • GatherMate - 采集助手
  • GearScore - 显示装备的GS评分
  • Grid - 团队战斗模块
  • ItemDB - 内置物品缓存查询数据库
  • JPack - 背包和银行一键整理
  • LootFilter - 物品拾取过滤
  • Mappy - 小地图美化
  • Mapster - 大地图增强
  • MobInfo - NPC信息增强插件
  • oCC - 法术冷却时间计时和触发技能提醒
  • Omen - 仇恨统计插件
  • Outfitter - 一键换装备
  • Parrot - 战斗信息增强
  • Perl - 头像增强
  • Postal - 邮件功能增强
  • Prat3 - 聊天窗口增强插件
  • Quartz3 - 施法动作条增强
  • RatingBuster - 技能等级和百分比转换
  • Recounter - 团队伤害统计
  • ShadowedUF - 3D头像美化
  • Skada - 团队伤害统计
  • SpellID - 查看技能和法术ID
  • TinyPad - 游戏内记事本插件
  • Titan - 信息条面板
  • TradeskillInfo - 商业技能资料库
  • VendOmatic - 自动出售, 补充物品
  • WebDKP
  • WoWPro - 专业玩家升级建议指引
  • NPCScan - 稀有怪扫描

注: 在单机版情况下通常我会禁用:

  • LootFilter: 很少会大量Farmer巨量物品
  • Recounter: 有Skada插件作为替代
  • Grid: 不是治疗不打团队副本, 不需要团队战斗框架
  • Perl, ShadowedUF:
  • WebDKP: 单机不使用DKP系统
