燃烧的远征声望攻略¶

简要说明¶

在燃烧的远征资料片中, 最重要的声望是五大头部附魔声望: 荣誉堡(联盟)/索尔玛(部落)(对应治疗职业), 塞纳里奥远征队(对应物理DPS职业), 撒塔(对应法系DPS职业), 贫民窟(台湾翻译陰鬱城)(对应猎人萨满), 时光守望者(对应Tank职业), 和所有职业肩膀附魔的 奥多尔 和 占星者

五大头部附魔声望出售的其他重要物品有:

  • 尊敬可获得各声望对应的副本群的英雄难度的钥匙
  • 对应各职业的头部附魔
  • 商业技能高级配方

具体冲各个声望的方法是如下:

对于五大头部附魔声望, 从 友善到崇敬 都可以通过 刷对应普通副本和做任务 来提升声望。而从 崇敬到崇拜 只能通过 刷英雄副本 来提升。

奥多尔和占星者¶

对应的两个阵营, 类似凄凉的半人马, 只能选择其一（有更改的方法, 不过非常麻烦, 基本上选了就是一辈子的事情了）。奥尔多是一群德莱尼祭祀, 当年沙特拉斯被屠城的时候他们躲了起来, 在感应到纳鲁降临后, 他们又回到了沙特拉斯。占星者直译是占卜者, 他们是在沙特拉斯重建后凯尔王子排来攻城的血精灵, 但在城下却集体放下武器倒戈——因为他们获知了凯尔的阴谋, 觉得只有纳鲁和圣光才是拯救血精灵一族的出路。两派都很忠心纳鲁, 但互相看不顺眼, 奥尔多觉得占星者有叛徒的前科信不过; 占星者则受不了奥尔多总以正统纳鲁信徒的调调自居。奥尔多主要集中力量打击燃烧军团, 占星者则集中力量打击阿蛋的血精灵军团。

奥多尔和占星者两者的肩膀附魔差别不大, 但是商业技能图纸完全不同。粗略概括的话: 占星者偏PVP, 奥抗自然抗装, Tank; 奥尔多偏PVE, 火抗暗抗装, 物理dps。

两个声望只能选择一个声望冲, 因为冲一边的声望会同时降低另一边的声望。当然冲完一边之后可以在撒塔斯城下城区找一个NPC接可重复性的任务, 可以将已经仇恨的声望冲回中立。

  • 奥多尔: 从 低等级/高等级 军团恶魔身上获得 基尔加丹印记/萨格拉斯之印, 或 军团装备 上交给NPC
  • 占星者: 从 低等级/高等级 血精灵身上获得 火翼纹章/日怒徽记, 或 奥术宝典 上交给NPC

改换阵营:

奥尔多 -> 占星者

想从奥尔多阵营改换到占星者的玩家需要到贫民窟找奥术师艾蒂莉亚。她会交给玩家一个可重复的任务 上交湿鳞蜥蜴的眼睛。玩家可以在泰罗卡森林的蜥蜴身上打到湿鳞蜥蜴的眼睛。每8个泰罗卡森林的蜥蜴眼睛换250该阵营的声望。这是将玩家的占星者声望提升到 中立 的唯一方法。

占星者 -> 奥尔多 想从占星者阵营改换到奥尔多的玩家需要到贫民窟找莎妮尔。她会交给玩家一个可重复的任务 上交湿鳞蜥蜴的眼睛。玩家可以在泰罗卡森林的蜥蜴身上打到湿鳞蜥蜴的眼睛。每8个泰罗卡森林的蜥蜴眼睛换250该阵营的声望。这是将玩家的奥尔多声望提升到 中立 的唯一方法。
